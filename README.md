# AI Decision Copilot

A predictive analytics and decision-support interface for Finance, Ops, and Growth managers.

## Features
- **Data Ingestion**: Drag & drop CSV/Excel upload with automatic health checks.
- **Smart Analysis**: AI-driven "Must-Ask" questions to uncover hidden variance.
- **Scenario Builder**: Interactive sliders to model "What-if" outcomes (e.g., Hiring Freeze impact).
- **Decision Memo**: One-click generation of exec-ready recommendation documents.

## Tech Stack
- **Framework**: React 18 + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS v3.4 (Dark Mode optimized)
- **Charts**: Recharts
- **Icons**: Lucide React

## Getting Started

1.  **Install dependencies**
    ```bash
    npm install
    ```

2.  **Start local server**
    ```bash
    npm run dev
    ```

3.  **Build for production**
    ```bash
    npm run build
    ```

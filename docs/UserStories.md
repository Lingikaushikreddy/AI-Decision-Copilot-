# User Stories

## Epic: Financial Analysis (Finance Manager)
*   **Story 01**: As a Finance Manager, I want to upload my quarterly budget Excel sheet so that I can instantly identify categories with >10% variance using natural language.
*   **Story 02**: As a Finance Manager, I want to generate a "Budget Variance Memo" that synthesizes my raw data into a narrative explanation for the CFO, so I can save 4 hours of writing time.
*   **Story 03**: As a Finance Manager, I want to ask "What happens if we freeze hiring?" and see a projection of Q4 OpEx, so I can evaluate cost-saving measures.

## Epic: Operational Efficiency (Ops Manager)
*   **Story 04**: As an Ops Manager, I want to connect my inventory database so that I can query current stock levels across all 5 warehouses without writing SQL.
*   **Story 05**: As an Ops Manager, I want the system to recommend inventory transfers to balance stock levels, identifying locations with excess vs. deficit.
*   **Story 06**: As an Ops Manager, I want to view a map visualization of my delivery fleet's current bottlenecks to make rerouting decisions.

## Epic: Market Expansion (Growth Manager)
*   **Story 07**: As a Growth Manager, I want to compare CAC (Customer Acquisition Cost) across 3 different marketing channels to decide where to double-down our spend.
*   **Story 08**: As a Growth Manager, I want to simulate the revenue impact of a price increase for the Enterprise tier, so I can assess the risk of churn vs. revenue gain.

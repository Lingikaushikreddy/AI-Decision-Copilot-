# Product Roadmap: AI Decision Copilot

## Phase 1: MVP (Minimum Viable Product)
**Focus**: Single Persona (Finance) & Core Data Ingestion.

*   **Features**:
    *   Ingest CSV/Excel files (Static data).
    *   Basic Natural Language Query interface ("Why is Q3 budget off?").
    *   Generate simple Text Summary outputs.
    *   Citations: Link back to specific cells/rows in source data.
*   **Goal**: Validate that the LLM can accurately interpret financial tabular data.

## Phase 2: V1 (Market Entry)
**Focus**: Multi-Persona support & Structured Outputs.

*   **Features**:
    *   **Decision Memo Generator**: Standardized templates for Finance, Ops, and Growth.
    *   **Scenario Modeling**: Basic "What-if" parameters sliders (e.g., +/- 10% revenue).
    *   **Visualizations**: Auto-generate Bar/Line charts for key trends.
    *   **Connectors**: Google Sheets & SQL integration (Read-only).
*   **Goal**: Reach "Product-Market Fit" by solving end-to-end decision workflows.

## Phase 3: V2 (Scale & Intelligence)
**Focus**: Predictive Capabilities & Collaboration.

*   **Features**:
    *   **Proactive Insights**: System alerts user to anomalies *before* they ask.
    *   **Multi-modal inputs**: Ingest PDFs, Slide decks, and Images.
    *   **Collaboration**: Shared workspaces to comment on and edit Decision Memos.
    *   **Predictive Analytics**: Forecasting models integrated into scenarios.
*   **Goal**: Become the default "OS" for strategic planning.
